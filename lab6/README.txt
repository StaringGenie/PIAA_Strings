*Лабораторная работа №6. Алгоритм Ахо-Корасика*

>> Алгоритм сборки:
1) Перейти в папку lab6, прописав cd lab6
2) Прописать sudo chmod +x build.sh
3) Затем прописать ./build.sh
4) Для запуска программ перейти в папку build, прописав cd build

>> Алгоритм Ахо-Корасик:
1) Находясь в папке build написать ./AKSearch
2) Ввести текст, в котором будет произведен поиск
3) Ввести цифру - число шаблонов которые будут введены в дальнейшем
5) Ввести шаблоны для поиска в строке
4) Результатом будут являться индексы вхождений шаблонов в текст сопоставленные номерам шаблонов

>> Поиск строки "джокера" в тексте:
1) Находясь в папке build написать ./AKJokerSearch
2) Ввести первую строку текста
3) Ввести строку "джокер"
4) Ввести символ "джокер"
5) Результатом будут являться позиции вхождения строки "джокера" в текст

>> Вызов тестирования представленных алгоритмов:
(7 тест кейсов для стандартного алгоритма Ахо-Корасика, 9 кейсов для модифицированного под алгоритм поиска вхождения строки Джокера)
1) Находясь в папке build написать ./AKSearchTest для запуска тестирования стандартного алгоритма Ахо-Корасик.
2) Находясь в папке build написать ./AKJokerSearchTest для запуска тестирования алгоритма поиска строки "джокера".


>> Для очистки директории build от всех сгенерированных в процессе сборки файлов, за исключением Makefile:
1) Находясь в папке build написать ./clean.sh
